# trading-app

Simple Spring Boot starter project for TradingApp.

Requirements
- Java 17+
- Maven 3.6+

Build

Using a local Maven installation:

```powershell
mvn -DskipTests package
```

Run

```powershell
mvn spring-boot:run
```

Try the sample endpoint:

```powershell
curl http://localhost:8080/api/hello
```
